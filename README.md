# 📦 Procurement Strategy Predictor

Live Demo: https://procurement-strategy-predictor-vfnpkxaynblfqma9xxicsh.streamlit.app/

A Streamlit web application that predicts the **Kraljic Procurement Category** for products using a Machine Learning model.

This tool helps procurement professionals and supply chain analysts classify products into one of the four Kraljic matrix categories:

✔ Strategic  
✔ Leverage  
✔ Bottleneck  
✔ Non-Critical

---

## 🧠 About

This project uses a supervised ML model (Naive Bayes with feature scaling) trained on realistic procurement data to classify products based on:

- Lead Time
- Order Volume
- Cost per Unit
- Supply Risk Score
- Profit Impact Score
- Environmental Impact
- Single Source Risk

The app was built using **Streamlit**, making it easy to interact with the model directly through a UI.

🔗 Live App: https://procurement-strategy-predictor-vfnpkxaynblfqma9xxicsh.streamlit.app/

---

## 🚀 Features

✅ Enter product attributes  
✅ Get predicted Kraljic category  
✅ Clean and responsive Streamlit UI  
✅ Model saved with pickle for easy reuse  
✅ Professional machine-learning pipeline
